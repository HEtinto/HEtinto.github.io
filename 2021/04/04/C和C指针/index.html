1.1.1动态内存申请

  通常情况下，像int array[2][2] ={{1,2},{3,4}}这种方式所申请的数组内存是连续的。对于连续内存的数组而言，在进行数组内存复制等情况时通常只要复制一次，比较高效。
  但是也可以给数组分配不连续的内存地址,下面的示例展示了如何向数组分配不连续的内存地址： 它首先分配“外层”数组，然后分别用malloc语句为每一行分配内存(matrix[i] = (int*)malloc(columns * sizeof(int));为每个数组指针申请一行整型数组指针的大小)。

  运行结果：从运行结果可以看到，虽然同一行数组的不同位置的元素的地址时连续的，但是，不同行的数组的地址时不连续的，这就是动态申请不连续数组内存的方法。

  下面这个函数matrix_add(int arr1[ ][3], int arr2[ ][3])向我们只能展示了如何传递int[2][3]数组，这里第一维默认不需要给出。同时，该函数实现了两个矩阵相加，并且返回了一个可以索引的二维数组，这个二维数组的内存是动态申请的，申请的语句为：int** point_matrix = (int**)malloc(2 * sizeof(int*));。与上面函数中的形参int *arr相比，这里使用arr1[][3]的好处是可以将传进来的数组指针直接按照数组下标的方式进行引用。
