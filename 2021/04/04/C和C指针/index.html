  高维数组，数组中的第一维度的值可以不给出，后面维度的值必须给出。这里有两种定义的方式，下面两种方式是等价的：

  当我们向函数传递二维数组时，传递过去的实际是组成二维数组的一维数组的地址，该一维数组存储着同一行元素的地址，通过对一维数组的解引用地址，我们可以得到该一维数组上的各个元素的值，同时，用过第二维度的值，我们可以知道该一维数组的最大下标。所以我们很自然的取到该数组上的值，即先获取一维数组的地址，然后通过一维数组的地址，索引到该一维数组上其他元素的地址，最后对其它元素的地址进行解引用即可得到所要的地址。

  下面这个函数matrix_add(int arr1[ ][3], int arr2[ ][3])向我们只能展示了如何传递int[2][3]数组，这里第一维默认不需要给出。同时，该函数实现了两个矩阵相加，并且返回了一个可以索引的二维数组，这个二维数组的内存是动态申请的，申请的语句为：int** point_matrix = (int**)malloc(2 * sizeof(int*));。
