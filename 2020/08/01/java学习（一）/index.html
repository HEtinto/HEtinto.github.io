java学习笔记

1.基本类型、数组和枚举类型

1.1 标识符和关键字

  • 标识符由字母、下划线、美元符号和数字组成，长度不受限制
  • 标识符的第一个字符不能是数字字符
  • 标识符不能是关键字
  • 标识符不能是true、false和null（尽管true、false和null不是java关键字）
  • 标识符区分大小写

    1.2 基本数据类型

  • 逻辑类型：boolean
  • 整数类型：byte、short、int、long
  • 字符类型：char
  • 浮点类型：float、double

1.3从命令行输入与输出数据

Scanner是JDK 1.5新增加的一个类，可以使用该类创建一个对象：

Scanner reader = new Scanner(System.in);

然后reader对象调用nextBoolean()、nextByte()、nextShort()、nextInt()、nextLong()、
nextFloat()和nextDouble()，读取用户在命令行（例如，MS-DOS窗口）输入的各种基本类型数据。
上述方法在执行过程中都会发生阻塞，程序等待用户输入数据并按Enter键确认。

1.4 数组

  • 声明数组
  • 创建数组

为数组分配内存空间 ，数组索引从0开始，在访问数组元素时注意不要越界。

数组名称 = new 数组元素的类型[数组元素的个数]

  • length的使用

    对于一唯数组，”数组名字.length”的值就是数组中元素的个数；对于二维数组，”数组名字.length”的值是它含有的一唯数组的个数。

例1.4数组的使用

  • 数组复制

public static void arraycopy(sourceArray, int index1, copyArray, int index2, int length)

将数组sourceArray中的索引值从index1到index1+length-1复制到copyArray的第index2到第index2+length,如果数组copyArray不能存放下复制的数据，程序将引发异常。

public static double[] copyOf(double[] original, int newLength)

可以将参数original指定的数组中从索引0开始的newLength个元素复制到一个新的数组中，并返回这个新数组，且新数组的长度为newLength。

public static double[] copyOfRange(double[] original, int from, int to)

该方法可以把参数original指定的数组中从索引from至to-1的元素复制到一个新数组中，并返回这个新数组，即新数组的长度为to-from。

  • 排序与二分查找

Array调用方法： 实现快速排序，升序排序。

public static void sort(double a[])

将a指定的double类型数组中索引start至end-1的元素的值按照升序排序。

public static void sort(double a[], int start, int end)

二分查找

public static int binarySearch(double[] a, double number)

2.类和对象

2.1 类

  • 类的实现包括两部分，即类的声明和类体

  • 实例变量和类变量

    用关键字static声明的称为类变量或静态变量，否则称为实例变量，即需要创建实例才会拥有实际空间的变量。

  • 成员变量的隐藏

    如果局部变量的名字与成员变量的名字相同，那么成员变量被隐藏，即这个成员变量在这个方法内暂时失效。

    2.2 对象

  • 对象的声明

类的名字 对象名称

  • 对象声明后，需要为对象分配内存空间

已经声明的对象名称 = new 类名称()

  • 对象的内存模型

    在执行分配内存的语句时，系统会先给类中的成员变量分配内存空间，然后执行构造函数，若声明成员变量或构造函数内没有对
    变量指定初值，那么系统会赋予成员变量默认值。同时，会给出一个信息，来确保对象的成员变量的归属性，它返回一个引用给对象，
    使得对象能对自己的成员变量进行操作。

    2.3 对象的引用和实体

    java的对象是在堆(heap)中分配内存，对象的引用是在栈(stack)中分配内存。java拥有垃圾回收机制，它会周期性地
    检测某个实体是否已经不再被任何对象所引用，若是发现这样的实体，就释放实体的内存。

    2.4 对象的组合

    可以在一个类中声明其他类的对象，以此达到组合对象的目的。

    2.5 可变参数

    2.6 static 关键字

    类中的类变量不需要创建对象就会被分配内存空间，而实例变量在创建对象时分配内存空间，类变量的内存空间直到程序退出运行，才释放所占有的内存。
    当类的字节码文件被加载到内存时，类的实例方法不会被分配入口地址，在该类创建对象后，类的实例方法才被分配入口地址，且只分配一次，被所有的对象
    所共享，在该类再创建对象时，就不在分配入口地址。当所有对象都不存在时，方法的入口地址才被取消。对于类方法，在类的字节码被加载到内存中时就被
    分配了入口地址，可以直接通过类名称调用。类方法的入口地址直到程序退出才被取消。

    2.7 this 关键字

    this表示当前对象，通常使用this来表明当前对象，以区分局部变量防止混淆，因此当实例变量和局部变量名字相同时，this变量不可省略。

    2.8 包

    2.9 import 语句

    import语句用于引入类库中的类，基本语法格式如：

    import java.util.Data

    import java.util.*

    3.0 访问权限

    对于变量和函数的访问权限，只需要在创建时在前面加上相应的关键字即可，如：

    对于不用private,public,protected修饰的成员变量和方法称为友好变量和友好方法，对于友好变量和方法，只允许在同一个包中访问。
    如果用import语句引入了另一个包中的类，那么用该类创建的对象将不能访问自己的友好变量和友好方法。受保护的变量和方法具有和友好变量和方法
    相同的访问属性，即只能在同一个包中访问，但是它们之间也会有区别。

    3.1 自动装箱与拆箱

    类 Integer num = 100；等价于Integer number = new Integer(m);

    int x = num + num;等价于int x = number.intValue() + number.intValue();

    3.2 反编译器和文件生成器

    查看源码类中的public方法的名字和public成员变量的名字：javap java.util.Data

    查看类中的所有方法和成员变量：javap -privae java.util.Data

    使用javadoc Example.java在文件夹下生成多个HTML文档，查看这些文档可以知道源文件中类的组成结构，如类的方法和成员变量。
    同时也可以通过下面的方法指定生成目录：javadoc -d C:\document Example.java
